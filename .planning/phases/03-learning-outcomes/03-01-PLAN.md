   - `PoKey = 'po2' | 'po5' | 'po3'` 타입 export
   - `PoGradeDistribution` 인터페이스 export: { total, high, mid, low, achieveRate(소수점 1자리), achieved(boolean) }
   - `PoAchievementData` 인터페이스 export: Record<PoKey, Record<ClassCode | 'all', PoGradeDistribution>>

2. `computeDistribution(students, poKey)` 내부 함수:
   - students 배열과 poKey를 받아 PoGradeDistribution을 반환
   - total === 0 가드: { total: 0, high: 0, mid: 0, low: 0, achieveRate: 0, achieved: false } 반환
   - getPoGrade(s[poKey], PO_THRESHOLDS[poKey])로 각 학생 등급 카운트
   - achieveRate = Math.round(((high + mid) / total) * 1000) / 10
   - achieved = (high + mid) / total >= 0.8 (부동소수점 이슈 방지 -- achieveRate가 아닌 raw 비율로 판정)
   - import: getPoGrade, PO_THRESHOLDS from '@/lib/grade-parser', StudentGrade/ClassCode from '@/types/grades'

3. `computePoAchievement(students)` export 함수:
   - poKeys: ['po2', 'po5', 'po3'] 순회
   - classCodes: ['A', 'B', 'C'] + 'all'(전체)
   - 각 PO x 각 반 + 전체에 대해 computeDistribution 호출
   - PoAchievementData 반환

4. `getLowGradeStudents(students, poKey, classCode?)` export 함수:
   - classCode가 지정되면 해당 반만 필터, undefined면 전체
   - getPoGrade(s[poKey], PO_THRESHOLDS[poKey]) === '하' 인 학생 반환
   - StudentGrade[] 반환

주의사항:
- PO_THRESHOLDS 임계값을 하드코딩하지 말 것 -- 반드시 grade-parser.ts에서 import
- computeDistribution은 export하지 않음 (내부 함수)
- 주석은 한국어로 작성

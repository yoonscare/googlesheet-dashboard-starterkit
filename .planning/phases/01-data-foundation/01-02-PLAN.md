1. **getGradeData(): Promise&lt;GradeDataResult&gt;** — 메인 통합 함수:

   a. `isGoogleSheetsConfigured()` 체크 — false이면 `{ students: mockGradeData, dataSource: 'mock' }` 즉시 반환

   b. `Promise.all`로 3반 병렬 페칭:
      ```
      const DATA_RANGE = 'A4:AE';  // 4행부터 끝까지, 헤더 제외
      const [classARows, classBRows, classCRows] = await Promise.all([
        fetchSheetData(`A반!${DATA_RANGE}`).catch(() => null),
        fetchSheetData(`B반!${DATA_RANGE}`).catch(() => null),
        fetchSheetData(`C반!${DATA_RANGE}`).catch(() => null),
      ]);
      ```

   c. 각 반별 파싱 헬퍼:
      - rows가 null이면(해당 반 페칭 실패) `getMockForClass(classCode)` 호출하여 mock 대체
      - rows가 있으면 `rows.map(row => parseGradeRow(row, classCode)).filter(Boolean)` 파싱

   d. dataSource 결정:
      - 3반 모두 성공: `'sheets'`
      - 일부 실패: `'partial-mock'`
      - 전체 try-catch에서 에러 발생: `'mock'` (전체 폴백)

   e. 반환: `{ students: [...classA, ...classB, ...classC], dataSource }`

   f. catch 블록에서 `console.error('성적 데이터 페칭 실패, mock 데이터로 대체:', error)` 로그 후 `{ students: mockGradeData, dataSource: 'mock' }` 반환

2. Import 관계:

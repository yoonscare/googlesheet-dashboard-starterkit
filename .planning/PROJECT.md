# 간호학과 아동실습 성적 대시보드

## What This Is

25학년도 2학기 3학년 아동간호학 실습 성적 데이터를 Google Sheets에서 가져와, 교수들이 성적 분포·학생별 현황·반 간 비교·학습성과 달성도를 한 눈에 파악할 수 있는 웹 대시보드. 교수 여러 명이 공유하여 사용하며, 학생 이름은 익명 처리(김*훈)된다.

## Core Value

교수가 성적 데이터를 시각적으로 분석하여 공정한 상대평가(A 이상 40% 제한)와 학습성과 달성도 판단을 효율적으로 할 수 있어야 한다.

### Out of Scope

- 학생 직접 열람 기능 — 교수 전용 대시보드
- 성적 입력/수정 — Google Sheets에서 직접 관리
- 등급(A+/A/B+...) 자동 부여 — 교수 재량이므로 컷라인 참고용만 제공
- 모바일 앱 — 웹 대시보드로 충분

## Context

### 데이터 소스 구조 (Google Sheets)

**시트**: A반, B반, C반 (동일 구조)
**헤더**: 3행 (병합 셀 포함), 4행부터 학생 데이터
**학생 정보 열**: 열번호, 실습기간, 실습기관, 학번, 이름
**반당 학생 수**: 40명 이상

**성적 구성**:
| 영역 | 항목 | 배점 |
|------|------|------|
| 실습평가 (80점) | 사전학습평가 | 10 |
| | 보고서평가 | 30 |
| | 실습지도교수평가 | 20 |
| | 현장지도자평가 | 20 |
| 학습성과 (80점) | 대상자간호 (학습성과2) | 64 |
| | 안전과질 (학습성과5) | 8 |
| | 전문직 (학습성과3) | 8 |
| 출석 | 출석점수 | 20 |
| **총점** | | **100** |
| 핵심간호술 | 별도 평가 | 100 |

※ 실습평가 80점 + 학습성과 80점 = 160점이지만, 총점 산출 시 100점 만점으로 환산됨

### 평가 체계

**상대평가**:
- A 이상(A+, A): 반별 학생 수의 40%까지만 부여 가능
- A+/A 구분: 교수 재량
- B 이하: 나머지 60%, 교수 재량

**학습성과 달성도 기준** (간호교육인증평가):
- 상: 해당 학습성과 만점의 85% 이상
- 중: 해당 학습성과 만점의 60% ~ 85% 미만
- 하: 해당 학습성과 만점의 60% 미만

### 기술 환경

기존 프로젝트에 Next.js 16, React 19, Tailwind CSS v4, shadcn/ui, Recharts, Google Sheets API 연동이 이미 구축되어 있음. 기존 매출 대시보드 코드를 성적 대시보드로 교체.

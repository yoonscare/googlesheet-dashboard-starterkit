1. **lib/grade-histogram.ts 생성** — 순수 함수, 서버사이드:
   - `HistogramDataPoint` 인터페이스 export:
     ```typescript
     export interface HistogramDataPoint {
       range: string;  // "0-9", "10-19", ..., "90-100"
       A: number;      // A반 학생 수
       B: number;      // B반 학생 수
       C: number;      // C반 학생 수
     }
     ```
   - `buildHistogramData(students: StudentGrade[]): HistogramDataPoint[]` export:
     - 11개 구간 생성: 0-9, 10-19, ..., 80-89, 90-100 (마지막 구간은 90-100으로 11점)
     - 각 구간에서 반별 학생 수 카운트: `students.filter(s => s.classCode === cls && s.totalScore >= low && s.totalScore <= high).length`
     - **주의:** 점수가 정확히 경계(10, 20, 30...)인 경우 상위 구간에 포함되도록 `>= low && <= high` 사용. 이때 10-19와 20-29가 겹치지 않도록 low=i*10, high=(i===10 ? 100 : i*10+9).
   - `findCutlineRange(cutlineScore: number): string` export:
     - 컷라인 점수가 속하는 히스토그램 구간 라벨을 반환 (예: 73.5 → "70-79")
     - `Math.floor(cutlineScore / 10) * 10`으로 구간 하한 계산
     - 90 이상이면 "90-100" 반환
   - import: `import type { StudentGrade } from '@/types/grades'`
   - 주석 한국어

     - 전체 컷라인(전체 학생 기준이 아닌 반별)은 1개의 대표값으로 단순화 가능하나, DASH-03 요구사항이 "반별"이므로 각 반 컷라인 점수를 Tooltip 또는 차트 하단 범례 텍스트로 보조 표시.

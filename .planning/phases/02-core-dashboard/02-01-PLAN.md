   ```typescript
   export interface GradeKpiData {
     totalStudents: number;
     classAverages: Record<ClassCode, number>;  // 반별 평균 (소수점 1자리)
     overallAverage: number;                     // 전체 평균 (소수점 1자리)
     highest: number;
     lowest: number;
     stdDev: number;                             // 표준편차 (소수점 1자리)
     cutlines: Record<ClassCode, number>;        // 반별 40% 컷라인 점수
   }
   ```

2. **lib/grade-kpi.ts 생성** — 순수 함수, 서버사이드 계산:
   - `computeGradeKpi(students: StudentGrade[]): GradeKpiData` 함수 export
   - 반별 평균: `students.filter(s => s.classCode === cls)` 후 totalScore 평균. `Math.round(avg * 10) / 10`으로 소수점 1자리.
   - 전체 평균: 전체 학생 totalScore 평균. 동일 반올림.
   - 최고점/최저점: 빈 배열 가드 필수 (`if (scores.length === 0) return 0`). `Math.max(...scores)` 대신 `scores.reduce` 사용으로 빈 배열 -Infinity 방지.
   - 표준편차: `Math.sqrt(sum((x - mean)^2) / n)`, 소수점 1자리 반올림.
   - 40% 컷라인: 해당 반 totalScore 내림차순 정렬 후 `Math.ceil(n * 0.4) - 1` 인덱스의 점수. 빈 반이면 0.
   - `avg(arr: number[]): number` 내부 헬퍼 함수 (빈 배열 → 0).
   - import는 `import type { StudentGrade, ClassCode } from '@/types/grades'` 사용.
   - 주석은 한국어로 작성 (프로젝트 CLAUDE.md 규칙).

**주의:** Research Pitfall 5 — 빈 배열에서 Math.max(...[]) = -Infinity 방지. reduce 패턴 사용 또는 가드 추가.

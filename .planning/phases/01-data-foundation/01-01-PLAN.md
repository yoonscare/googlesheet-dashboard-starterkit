<task type="auto">
  <name>Task 2: 열 인덱스 상수 + 파서 + 익명화 + 학습성과 등급 유틸</name>
  <files>lib/grade-parser.ts</files>
  <action>
lib/grade-parser.ts 파일을 새로 생성한다. 이 파일은 Google Sheets 행 데이터를 StudentGrade 객체로 변환하는 모든 순수 함수를 포함한다.

1. **COL 상수 (열 인덱스 맵)** — `as const` 객체로 0-based 인덱스 정의:
   - A(0)=순번, B(1)=실습기간, C(2)=실습기관, D(3)=학번, E(4)=이름
   - 사전학습 소계: I열(8) — F~I 범위에서 I가 소계 위치 (CONTEXT.md "F-I: 사전학습 (10점 만점)" 참고, 소계가 마지막 열)
   - 보고서: J(9)셀프, K(10)케이스, L(11)교육인, M(12)체크, N(13)대상, O(14)소계
   - 실습지도교수: P(15)대상, Q(16)안전, R(17)전문, S(18)소계
   - 현장지도자: T(19)대상, U(20)안전, V(21)전문, W(22)소계
   - 학습성과: X(23)=po2, Y(24)=po5, Z(25)=po3
   - AA(26)=실습성적합계(80점), AB(27)=출석(20점)
   - 핵심간호술: AD(29) — AC(28)가 조정 열로 추정
   - 주석으로 "실제 시트 검증 후 인덱스 조정 필요" 경고를 COL 상수 위에 명시

2. **anonymizeName(name: string): string** — 한국어 이름 익명화:
   - 2글자: 성 + * (김수 -> 김*)
   - 3글자 이상: 성(첫글자) + * + 끝글자 (김현우 -> 김*우, 사공현우 -> 사*우)
   - 빈 문자열이나 1글자는 그대로 반환

3. **anonymizeStudentId(id: string): string** — 학번 익명화:
   - 7자리 이상: 앞4자리 + *** + 뒤2자리 (202000316 -> 2020***16)
   - 7자리 미만: 그대로 반환

4. **parseGradeRow(row: string[], classCode: ClassCode): StudentGrade | null** — 행 파서:
   - 이름(COL.NAME)이 빈 문자열이면 null 반환 (빈 행 스킵)
   - `getNum(idx)` 헬퍼: `Number(row[idx])`가 NaN이면 0 반환 (trailing empty cells 대응)
   - `totalScore`는 `getNum(COL.PRACTICE_TOTAL) + getNum(COL.ATTENDANCE)`로 계산 (시트 합계가 아닌 우리가 직접 합산 — 조정 열 무시)
   - 이름/학번은 `anonymizeName()`, `anonymizeStudentId()`를 호출하여 익명화 후 반환 — 원본은 여기서 폐기

5. **PO_THRESHOLDS 상수** — 학습성과별 임계값:
   - po2: { max: 64, high: 54.4, mid: 38.4 }
   - po5: { max: 8, high: 6.8, mid: 4.8 }
   - po3: { max: 8, high: 6.8, mid: 4.8 }

6. **getPoGrade(score: number, threshold: { high: number; mid: number }): PoGrade** — 등급 판정:
   - score >= high → '상', score >= mid → '중', otherwise → '하'

7. **isAchieved(students: StudentGrade[], poKey: 'po2' | 'po5' | 'po3'): boolean** — 달성 판정:
   - 해당 학습성과에서 중 이상 비율이 80% 이상이면 true
   - 빈 배열이면 false

모든 함수는 순수 함수로 구현하고, 외부 라이브러리 없이 네이티브 JavaScript/TypeScript만 사용한다. 코드 주석은 한국어, 변수명/함수명은 영어.
